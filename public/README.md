# Public Folder

This folder contains static assets that are served directly by Next.js.

## How to Add Your Resume

1. Place your resume PDF file in this folder
2. Name it `resume.pdf` (or update the link in `components/Hero.tsx`)
3. The file will be accessible at `/resume.pdf`

## Example

If you have a file named `resume.pdf` in this folder, it will be accessible at:
- `http://localhost:3000/resume.pdf` (development)
- `https://yourdomain.com/resume.pdf` (production)

## Other Static Assets

You can also add:
- Images (logos, icons, etc.)
- Fonts
- Other static files

All files in this folder are served from the root URL path.
